## PLACE YOUR SLIDE FILES IN THIS FOLDER
=======================================
```
  Slide1.png         <- Introduction screen  (MUST be PNG)
  Slide2.png/.m4v    <- Reading page 1       (PNG or video)
  Slide3.png/.m4v    <- Reading page 2       (PNG or video)
  ...
  SlideN.png         <- Ending screen        (MUST be PNG)
```
Rules:
  1. Slide1 and SlideN must be .png or jpg
  2. All slides in between can be .png or .m4v (mix freely)
  3. Files must be consecutively numbered — no gaps,
  4. Example Slide1 to SlideXX where XX are numeric numbers
  5. Do not prefix numbers with a zero. **~~Slide06.png~~** is wrong. It should be Silde6.png
